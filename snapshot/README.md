# Torres Quintero Site Snapshot

This directory contains a migration snapshot of the public Wix site at `https://www.torresquintero.edu.mx` captured on 2026-05-24.

## What was captured

- Raw public page HTML in `raw/pages/`
- Wix sitemap files in `raw/sitemaps/`
- Link and media inventories in `inventory/`
- Original Wix media files referenced by the downloaded pages in `assets/originals/`

## Inventories

- `inventory/page-links.txt`: unique internal page URLs found in the downloaded HTML
- `inventory/transformed-media-urls.txt`: Wix image URLs as rendered on the site
- `inventory/original-media-files.txt`: original Wix media file ids used to download local copies
